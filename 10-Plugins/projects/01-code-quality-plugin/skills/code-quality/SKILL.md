# Code Quality Analysis

This skill provides comprehensive code quality analysis capabilities.

## Capabilities

### Security Analysis
Identify common security vulnerabilities:
- SQL injection
- Cross-site scripting (XSS)
- Authentication issues
- Sensitive data exposure
- Command injection
- Path traversal

### Code Quality
Evaluate overall code quality:
- Code complexity
- Code duplication
- Naming conventions
- Code organization
- Error handling
- Documentation

### Performance
Identify performance issues:
- Algorithm efficiency
- Resource usage
- Database query optimization
- Caching opportunities

### Best Practices
Check for best practices:
- Language-specific patterns
- Framework conventions
- Testing coverage
- Code style

## Usage

When analyzing code quality:

1. **Identify the scope**: What code to analyze
2. **Run analysis**: Check each category
3. **Prioritize findings**: Sort by severity
4. **Provide suggestions**: Actionable fixes

## Output Format

```markdown
## Code Quality Report

### Summary
Brief overview of findings.

### Issues

#### High Severity
- [Issue description]

#### Medium Severity
- [Issue description]

#### Low Severity
- [Issue description]

### Recommendations
1. [Recommendation]
2. [Recommendation]
```
