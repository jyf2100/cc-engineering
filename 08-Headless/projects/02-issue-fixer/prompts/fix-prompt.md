# Issue 修复 Prompt 模板

## Bug 修复

```
Fix the following bug:

**Title**: {{issue_title}}
**Description**: {{issue_body}}

### Steps to fix:
1. Understand the bug from the description
2. Locate the relevant code files
3. Identify the root cause
4. Implement a minimal fix
5. Ensure no regressions

### Constraints:
- Only modify files in src/ and tests/
- Do not change configuration files
- Keep changes minimal and focused
- Add comments explaining the fix if necessary
```

## 功能实现

```
Implement the following feature request:

**Title**: {{issue_title}}
**Description**: {{issue_body}}

### Implementation steps:
1. Understand the requirements
2. Design the solution
3. Implement the feature
4. Add appropriate tests
5. Update documentation if needed

### Constraints:
- Follow existing code patterns
- Maintain backward compatibility
- Include error handling
- Keep the implementation simple
```

## 性能优化

```
Optimize the code for the following performance issue:

**Title**: {{issue_title}}
**Description**: {{issue_body}}

### Optimization approach:
1. Profile and identify bottlenecks
2. Analyze the problematic code
3. Propose optimization strategies
4. Implement the optimization
5. Verify improvement

### Constraints:
- Maintain existing functionality
- Document any trade-offs
- Include benchmarks if possible
```
